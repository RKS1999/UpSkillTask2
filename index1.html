Selectors in HTML

Element Selector

The element selector selects HTML Elements based on the element name.

ID Selector

The ID selector uses the id attribute of an HTML element to select a specific element. The ID of an element is unique within a page, so the id selector is used to select one wnique element!

Class Selector

The class selector selects HTML elements based on the class attribute. The class attribute is used to assign multiple classes to an HTML element. The class selector is used to select multiple wnique elements!
